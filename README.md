## Tech Stack

- **Framework** : Next.js

- **Styling**: Tailwind CSS

## Installation

**Install my-project with npm:**

```bash
  git clone https://github.com/aayman01/digital_agency.git
  cd digital_agency
```
**Install dependencies:**
```bash
npm install
```
**To run the development server**
```bash
npm run dev
```
